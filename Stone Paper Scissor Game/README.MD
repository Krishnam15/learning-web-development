# Stone Paper Scissors Game

## Overview
A simple browser-based implementation of the classic Stone Paper Scissors game.  
This project was built to practice core JavaScript concepts such as event handling, conditional logic, and DOM manipulation.

## Features
- User vs Computer gameplay
- Randomized computer choices
- Real-time result display
- Simple and responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## How It Works
- The user selects Stone, Paper, or Scissors
- The computer randomly generates a choice
- Game logic determines the winner
- The result is displayed instantly on the UI

## Project Structure
- `index.html` – UI structure
- `style.css` – Styling and layout
- `app.js` – Game logic and event handling
- `images/` – Assets used in the UI

## Learning Outcomes
- DOM manipulation using JavaScript
- Event-driven programming
- Writing clean conditional logic
- Structuring a small frontend project

## Status
Completed basic version. Can be extended with score tracking or animations.
