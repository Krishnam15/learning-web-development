# Learning Web Development

## Overview
This repository contains a collection of small web development projects created as part of my learning and practice in frontend development.  
Each project focuses on strengthening core concepts such as HTML, CSS, JavaScript, and React through hands-on implementation.

## Projects Included

### 1. React Weather App
A React-based application that fetches and displays real-time weather data using a third-party API.
- Concepts: React components, API integration, state management
- Tech: ReactJS, JavaScript, Axios

### 2. Stone Paper Scissors Game
A simple browser-based game implementing core JavaScript logic and DOM manipulation.
- Concepts: Conditional logic, event handling
- Tech: HTML, CSS, JavaScript

### 3. Tic Tac Toe Game
An interactive Tic Tac Toe game built using vanilla JavaScript.
- Concepts: Game logic, DOM manipulation, state handling
- Tech: HTML, CSS, JavaScript

### 4. Apni Kaksha
A static courses platform web page featuring multiple courses with embedded videos and linked PDF learning resources.
- Concepts: HTML structuring, CSS layout, and content organization.
- Tech: HTML, CSS.
